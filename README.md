# 🧑🏻‍💻 Rime 配置

## 文件说明

```bash
.
 📝 README.md                                 # 使用说明书
 📄 custom_phrase.txt                         # 配置文件 - 自定义短语，不适合导入大量条目
 📄 default.custom.yaml                       # 配置文件 - 自定义一些输入法的功能（全局设定）
 📁 dict/                                     # 词库目录
    📖 dict/luna_pinyin.hanyu.dict.yaml       # 词库文件 - 现代汉语常用词表
    📖 dict/luna_pinyin.i.dict.yaml           # 词库文件 - 用户私人词库
    📖 dict/luna_pinyin.name.dict.yaml        # 词库文件 - 人名（用户词库）
    📖 dict/luna_pinyin.zhwiki.dict.yaml      # 词库文件 - 中文维基百科词条
    📖 dict/sogou_pinyin.idiom.dict.yaml      # 词库文件 - 成语俗语（搜狗词库）
    📖 dict/sogou_pinyin.newwords.dict.yaml   # 词库文件 - 网络流行新词（搜狗词库）
    📖 dict/sogou_pinyin.poem.dict.yaml       # 词库文件 - 古诗词（搜狗词库）
 📄 essay.txt                                 # 配置文件 - 八股文词频库
 📄 luna_pinyin.custom.yaml                   # 配置文件 - 朙月拼音用户设定
 📄 luna_pinyin.extended.dict.yaml            # 词库文件 - 明月拼音扩展词库
 📁 opencc/                                   # OpenCC（开放中文转换）目录
    📄 opencc/emoji_word.txt                  # 配置文件 - Rime Emoji 条目
    📄 opencc/emoji.json                      # 配置文件 - Rime Emoji 配置
 📄 rime.lua                                  # 配置文件 - 可以输出系统变量的函数
 📄 squirrel.custom.yaml                      # 配置文件 - 鼠须管（macOS）界面配置
```
